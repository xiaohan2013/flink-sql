# flink-sql
flink-sql相关功能验证

* flink 1.14.6 与 hudi 0.12.1 可用

## Hudi主要操作
* Query
* 快照， 增量， 读优化
* Stream Query, 读取实时变更记录
* Snapshot Query，读取最小快照
* ReadOptimized Query，仅读取列数据
* Incremental Query，读取指定历史时间段内提交的数据
